# Navy Blue Music Player

A sleek, responsive audio player built with vanilla JS, HTML, & CSS. Personalized with soft blue (#c9d2fc) accents on a navy canvasâ€”perfect for late-night vibes.

## Features
- Playlist sorting & shuffling
- Play/pause/next/prev/delete tracks
- Seek bar with time display
- Mobile-responsive (try pinching on phone)
- Quincy Larson's freeCodeCamp tracks (or swap yours)

## Quick Start
- Clone: `git clone [your-repo-url]`
- Open `index.html` in browser (no server needed!)
- Or deploy to GitHub Pages for live demo.

## Tech
- Vanilla JS for logic
- CSS vars for easy theming
- Accessibility-ready (aria-labels, focus outlines)
